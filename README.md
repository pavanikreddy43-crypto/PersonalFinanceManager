# Personal Finance Manager

A command-line Personal Finance Manager built using Python that helps users manage income and expense transactions. The application uses Object-Oriented Programming (OOP), JSON for persistent storage, and input validation to provide a simple finance management experience.

---

## Features

- Add Income
- Add Expense
- View Transactions
- Edit Transactions
- Delete Transactions
- Dashboard Summary
- Persistent JSON Storage
- Input Validation
- Regular Expression Validation
- Object-Oriented Design

---

## Technologies Used

- Python
- JSON
- Object-Oriented Programming
- Regular Expressions
- File Handling

```bash

```

```bash
python main.py
```
